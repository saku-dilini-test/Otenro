Shipping Settings

Create a shipping option

How to create a shipping option

  1. Navigate to Commerce
  2. Select “Shipping”
  3. Click on "Add Shipping Options"
  4. Select “Flat Rate”, “Weight Based”, or “Pick Up”

Define Flat Rate

How to define Flat Rate

      What is Flat Rate shipping?
      Flat Rate Shipping allows you to define a standard rate per item and or per order.

      What is a per order fee?
      A fee that is applied to each order placed.

      What is a per item fee?
      A fee per item in an order that is placed.

    1. Navigate to Commerce
    2. Select “Shipping”
    3. Select “Flat Rate”
    4. Enter option name. This is the name that will be displayed to your customer when they select what shipping option they would like.
    5. Enter Per Order Fee and/or Per Item Fee
    6. Click on “Next”
    7. Select either “Ship to rest of the world” or “Countries Allowed”
    8. If you selected the “Countries Allowed” option, then you will need to select the list of countries that this option is available for

Edit Weight Based

For Weight Based

    What is Weight based shipping?
    Weight based shipping allows you to define a rate that varies based on the total order weight.

    1. Navigate to Commerce
    2. Select “Shipping”
    3. Select "Weight Based"
    4. Enter the Option Name. This is the name that will be displayed to your customer.
    5. Enter the first weight range. E.g. 0 kg to 0.5 kg
    6. Enter the cost in the corresponding cost field to the right of the weigh ranges
    7. Once you’re done with adding the weight ranges click on “Next”
    8. You will be navigated to "Country Restrictions"
    9. Select the list of countries that this option is available for
    10. When you are done Click “Save”
    11. If you would like to add additional weight ranges, please click on the “+” icon

Setting Up For Pick Up Request

How to Set Up For Pick Up Request

    What is Pick Up?
    Locations where your customer can pick up the order from

  1. Navigate to Commerce
  2. Select “Shipping”
  3. Select "Pick Up"
  4. Enter a location name. This location name will be displayed to your customer at check out.
  5. Enter the pick up location address
  6. You can enter a cost if you would like your customer to be charged a fee for pick up
  7. When you are done Click “Save”
