Order Settings

View an order

How to view an order

  1. Navigate to “Commerce”
  2. Select “Orders”
  3. You can search for a particular order by using the search bar or by going through the order list.
  4. Click on the order you would like to view details for

View fulfilled orders

How to view fulfilled orders

  2. Select “Orders”
  3. Select “Orders Fulfilled” from the tabs
